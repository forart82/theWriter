# Project rules & commands

## Commands

- Run dev server: `npm run dev`
- Build production bundle: `npm run build`
- Run linter checks: `npm run lint`

## Git Workflow & Rules

- **Pull before starting**: Always run `git pull --rebase` before writing any code.
- **Direct main commits**: Always commit and push directly to `main`.
- **Completion Rule**: Always commit and push all changes every time you have finished a task.
